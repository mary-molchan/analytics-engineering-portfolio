# Analytics Engineering Portfolio

Ce repository regroupe différents projets techniques et analytiques réalisés dans une logique de portfolio professionnel.  
L’objectif est de présenter ma capacité à transformer des besoins métier en solutions data structurées, documentées et exploitables.

---

## 🎯 Objectif du portfolio

Ce portfolio a pour objectif de démontrer mes compétences en :

- analyse de données ;
- préparation, nettoyage et transformation de données ;
- modélisation SQL et Snowflake ;
- création de dashboards Power BI ;
- automatisation de workflows analytiques ;
- documentation technique et fonctionnelle ;
- machine learning appliqué ;
- utilisation de l’IA générative pour améliorer les processus analytiques et organisationnels.

## 🧩 Approche projet

Chaque projet est structuré autant que possible avec :

- un contexte métier clair ;
- des objectifs définis ;
- une méthodologie documentée ;
- du code organisé ;
- des livrables exploitables ;
- des résultats interprétés ;
- des pistes d’amélioration ;
- une documentation technique ou fonctionnelle.

L’objectif n’est pas seulement de présenter du code, mais de montrer une démarche complète :  
**comprendre le besoin, structurer les données, produire une solution fiable et communiquer les résultats.**
